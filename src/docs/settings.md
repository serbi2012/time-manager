# 설정

**설정** 모달에서는 데이터 관리, 단축키 설정, 자동완성 옵션을 관리할 수 있습니다.

---

## 설정 열기

| 방법                         | 설명                       |
| ---------------------------- | -------------------------- |
| **설정 버튼**                | 헤더 우측의 ⚙️ 아이콘 클릭 |
| **[단축키](wiki:shortcuts)** | `Alt + ,`                  |

---

## 설정 모달 구성

설정 모달은 다섯 개의 탭으로 구성됩니다:

| 탭           | 주요 기능                       |
| ------------ | ------------------------------- |
| **테마**     | 헤더 색상 테마 설정             |
| **데이터**   | 프리셋 설정, 내보내기, 가져오기 |
| **자동완성** | 옵션 숨기기/복원                |
| **단축키**   | 단축키 활성화/비활성화          |

---

## 테마 탭

앱 전체의 테마 색상을 선택합니다. 헤더, 버튼, 링크 등 주요 UI 요소의 색상이 변경됩니다.

### 테마 색상 옵션

| 테마              | 적용 영역                       |
| ----------------- | ------------------------------- |
| **파란색 (기본)** | 헤더, 버튼, 링크, 스위치, 탭 등 |
| **초록색**        | 〃                              |
| **보라색**        | 〃                              |
| **빨간색**        | 〃                              |
| **주황색**        | 〃                              |
| **청록색**        | 〃                              |
| **검정색**        | 〃                              |

> **참고**: 테마 설정은 자동으로 저장되며, 로그인한 경우 다른 기기에서도 동일한 테마가 적용됩니다.

## 데이터 탭

프리셋 설정, 점심시간, 데이터 관리 기능을 제공합니다.

아래는 데이터 관리 패널의 모습입니다:

:::demo SettingsPanel:::

### 점심시간 설정

간트차트와 작업 시간 계산에 사용되는 점심시간을 설정합니다.

| 항목          | 기본값 | 설명          |
| ------------- | ------ | ------------- |
| **시작 시간** | 11:40  | 점심시간 시작 |
| **종료 시간** | 12:40  | 점심시간 종료 |

> **참고**: 점심시간은 작업 시간 계산에서 자동으로 제외됩니다. 간트차트에서도 회색 영역으로 표시됩니다.

### 작업 추가 시 구분자(postfix) 사용

[프리셋](wiki:work-preset)으로 작업을 추가할 때 거래명 뒤에 시간 구분자를 붙일지 설정합니다.

| 설정           | 동작             | 예시                   |
| -------------- | ---------------- | ---------------------- |
| **OFF** (기본) | 이름 그대로 추가 | `회의`                 |
| **ON**         | 시간 구분자 추가 | `회의_0122_093045_123` |

> **참고**: 같은 이름의 미완료 작업이 이미 있으면 자동으로 번호가 붙습니다.
> 예: `회의` → `회의 (2)` → `회의 (3)`

### 데이터 내보내기

모든 작업 기록과 설정을 **JSON 파일**로 내보냅니다.

| 항목            | 내용                                      |
| --------------- | ----------------------------------------- |
| **파일명**      | `time-manager-backup-YYYY-MM-DD.json`     |
| **포함 데이터** | records, templates, timer, custom_options |

### 데이터 가져오기

백업한 JSON 파일에서 데이터를 복원합니다.

> **⚠️ 주의**: 가져오기를 하면 **현재 데이터가 덮어씌워집니다**. 필요한 경우 먼저 내보내기를 해두세요.

### 서버에서 새로고침

로그인 상태에서 서버의 최신 데이터를 가져옵니다.

| 상태       | 동작                               |
| ---------- | ---------------------------------- |
| **로그인** | 서버에서 최신 데이터를 새로 불러옴 |
| **게스트** | 버튼 비활성화                      |

> **참고**: 데이터 변경 시 자동으로 서버에 저장되므로 일반적으로 새로고침이 필요하지 않습니다. 다른 기기에서 변경한 내용을 확인할 때 사용하세요.

### 트랜지션 효과

페이지 진입 시 UI 요소들이 부드럽게 나타나는 애니메이션 효과를 설정합니다.

| 설정                       | 설명                                      |
| -------------------------- | ----------------------------------------- |
| **페이지 진입 애니메이션** | 트랜지션 효과 활성화/비활성화 (기본: OFF) |
| **애니메이션 속도**        | 느리게 / 보통 / 빠르게 중 선택            |

> **참고**: 트랜지션은 데이터 로딩 완료 후 헤더, 사이드바, 간트차트, 작업 기록 테이블이 순차적으로 나타나는 효과입니다.

키보드 [단축키](wiki:shortcuts)를 관리합니다.

### 단축키 목록

모든 단축키가 테이블 형태로 표시됩니다:

:::demo ShortcutsTable:::

### 단축키 활성화/비활성화

각 단축키의 **스위치를 토글**하여 개별적으로 활성화/비활성화할 수 있습니다.

> **활용**: 브라우저나 다른 앱의 단축키와 충돌하는 경우 비활성화

### 기본값으로 초기화

**"기본값으로 초기화"** 버튼을 클릭하면 모든 단축키가 기본 상태로 복원됩니다.

## 자동완성 탭

자동완성에 표시되는 옵션을 관리합니다.

### 자동완성 옵션 종류

| 필드                     | 데이터 소스             |
| ------------------------ | ----------------------- |
| **작업명**               | 기존 작업 기록 + 프리셋 |
| **업무명**               | 기존 작업 기록 + 프리셋 |
| **거래명**               | 기존 작업 기록 + 프리셋 |
| **프로젝트 코드**        | 기존 작업 기록 + 프리셋 |
| **업무명 Select 옵션**   | 기본값 + 사용자 추가    |
| **카테고리 Select 옵션** | 기본값 + 사용자 추가    |

### 옵션 숨기기

불필요한 옵션을 자동완성에서 숨길 수 있습니다:

1. 숨길 옵션 체크
2. **"선택 항목 숨기기"** 버튼 클릭
3. 자동완성 목록에서 해당 옵션이 사라짐

### 숨긴 옵션 복원

1. **"숨긴 항목"** 섹션 펼치기
2. 복원할 옵션 선택
3. **"선택 항목 복원"** 버튼 클릭

## 설정 동기화

### 동기화되는 설정

| 설정 항목                   | 동기화 |
| --------------------------- | :----: |
| 앱 테마 색상                |   ✅   |
| 단축키 활성화 상태          |   ✅   |

| 작업 프리셋 순서            |   ✅   |
| 트랜지션 효과 설정          |   ✅   |
