# 설정

**설정** 모달에서는 테마, 애니메이션, 데이터 관리, 자동완성 옵션, 단축키를 관리할 수 있어요.

---

## 설정 열기

| 방법                         | 설명                         |
| ---------------------------- | ---------------------------- |
| **설정 아이콘**              | 헤더 우측의 설정 아이콘 클릭 |
| **[단축키](wiki:shortcuts)** | `Alt + ,`                    |

---

## 설정 모달 구성

설정 모달은 좌측 세로 탭으로 구성된 넓은 레이아웃이에요. 각 탭은 카드 스타일 섹션으로 나뉘어져 있어요.

| 탭             | 아이콘              | 주요 기능                                 |
| -------------- | ------------------- | ----------------------------------------- |
| **테마**       | 팔레트 아이콘       | 앱 테마 색상 선택                         |
| **애니메이션** | 번개 아이콘         | 페이지 트랜지션 효과                      |
| **데이터**     | 데이터베이스 아이콘 | 시간 설정, 프리셋 설정, 내보내기/가져오기 |
| **자동완성**   | 목록 아이콘         | 자동완성 옵션 숨기기/복원                 |
| **단축키**     | 키 아이콘           | 단축키 활성화/비활성화, 키 조합 변경      |

---

## 테마 탭

앱 전체의 테마 색상을 선택해요. 헤더, 버튼, 링크 등 주요 UI 요소의 색상이 변경돼요.

### 테마 색상 옵션

| 테마              | 적용 영역                                  |
| ----------------- | ------------------------------------------ |
| **파란색 (기본)** | 헤더 그라데이션, 버튼, 링크, 스위치, 탭 등 |
| **초록색**        | 동일                                       |
| **보라색**        | 동일                                       |
| **빨간색**        | 동일                                       |
| **주황색**        | 동일                                       |
| **청록색**        | 동일                                       |
| **검정색**        | 동일                                       |

> **참고**: 테마는 자동 저장돼요. 로그인하면 모든 기기에 동일하게 적용돼요.

---

## 애니메이션 탭

페이지 진입 시 UI 요소들이 부드럽게 나타나는 애니메이션 효과를 설정해요.

### 트랜지션 효과

| 설정                       | 설명                                                        |
| -------------------------- | ----------------------------------------------------------- |
| **페이지 진입 애니메이션** | 페이지 로딩 후 UI가 슬라이드되며 나타나요 (Switch로 ON/OFF) |
| **애니메이션 속도**        | 느리게 / 보통 / 빠르게 중 선택                              |

> **참고**: 애니메이션을 끄면 속도 선택이 비활성화돼요. 트랜지션은 헤더, 사이드바, 간트차트, 작업 기록 테이블이 순차적으로 나타나는 효과예요.

## 데이터 탭

시간 설정, 프리셋 설정, 데이터 관리 기능을 제공해요.

아래는 데이터 관리 패널의 모습이에요:

:::demo SettingsPanel:::

### 점심시간 설정

간트차트와 작업 시간 계산에 사용되는 점심시간을 설정해요.

| 항목          | 기본값 | 설명          |
| ------------- | ------ | ------------- |
| **시작 시간** | 11:40  | 점심시간 시작 |
| **종료 시간** | 12:40  | 점심시간 종료 |

> **참고**: 점심시간은 간트차트에 표시되며 작업 시간 계산 시 자동으로 제외돼요. 시간은 5분 단위로 설정할 수 있어요.

### 프리셋 설정 (고유 식별자 자동 추가)

[프리셋](wiki:work-preset)으로 작업을 추가할 때 거래명에 타임스탬프 구분자를 붙일지 설정해요.

| 설정           | 동작                   | 예시                   |
| -------------- | ---------------------- | ---------------------- |
| **OFF** (기본) | 이름 그대로 추가       | `회의`                 |
| **ON**         | 타임스탬프 구분자 추가 | `회의_0122_093045_123` |

> **참고**: 같은 이름의 미완료 작업이 이미 있으면 자동으로 번호가 붙어요.
> 예: `회의` → `회의 (2)` → `회의 (3)`

### 데이터 내보내기

모든 작업 기록과 설정을 **JSON 파일**로 내보내요.

| 항목            | 내용                                      |
| --------------- | ----------------------------------------- |
| **파일명**      | `time-manager-backup-YYYY-MM-DD.json`     |
| **포함 데이터** | records, templates, timer, custom_options |

### 데이터 가져오기

백업한 JSON 파일에서 데이터를 복원해요.

> **주의**: 가져오기를 하면 **현재 데이터가 대체돼요**. 필요한 경우 먼저 내보내기를 해두세요.

안내: "JSON 파일로 데이터를 백업하거나 복원할 수 있습니다. 가져오기 시 기존 데이터가 대체됩니다."

### 저장소 상태

현재 데이터 저장소 상태가 표시돼요:

| 상태       | 표시              | 설명                                                     |
| ---------- | ----------------- | -------------------------------------------------------- |
| **로그인** | "클라우드 연결됨" | "모든 데이터가 자동으로 동기화됩니다"                    |
| **게스트** | "로컬 저장"       | "로그인하면 여러 기기에서 데이터를 동기화할 수 있습니다" |

## 단축키 탭

키보드 [단축키](wiki:shortcuts)를 관리해요.

### 단축키 목록

모든 단축키가 테이블 형태로 표시돼요:

:::demo ShortcutsTable:::

### 테이블 컬럼

| 컬럼         | 설명                             |
| ------------ | -------------------------------- |
| **기능**     | 단축키가 수행하는 동작           |
| **단축키**   | 현재 설정된 키 조합              |
| **카테고리** | 일반, 타이머, 네비게이션, 데이터 |
| **활성화**   | Switch로 ON/OFF 토글             |

### 단축키 키 조합 변경

각 단축키의 키 조합을 원하는 대로 변경할 수 있어요:

1. 변경하고 싶은 단축키 클릭
2. 키 편집 모달이 열리면 **"여기를 클릭하고 단축키를 누르세요"**
3. 새 키 조합 입력 (예: `Ctrl+K`, `Alt+M` 등)
4. **"저장"** 클릭

> **제약 사항**:
>
> -   Ctrl/Alt/Shift 중 하나 이상의 수정자 키가 필요해요 (F1~F12 제외)
> -   다른 단축키와 중복 불가
> -   ESC로 취소

### 기본값으로 초기화

**"기본값으로 초기화"** 버튼을 클릭하면 모든 단축키가 기본 상태로 복원돼요.

> **모바일**: 단축키 설정은 PC에서만 사용할 수 있어요. 모바일에서는 "PC에서만 사용 가능" 안내가 표시돼요.

## 자동완성 탭

자동완성에 표시되는 옵션을 관리해요.

### 자동완성 옵션 종류

**입력 필드 자동완성** (4개):

| 필드              | 아이콘      | 데이터 소스             |
| ----------------- | ----------- | ----------------------- |
| **작업명**        | 파일 아이콘 | 기존 작업 기록 + 프리셋 |
| **업무명**        | 팀 아이콘   | 기존 작업 기록 + 프리셋 |
| **거래명**        | 쇼핑 아이콘 | 기존 작업 기록 + 프리셋 |
| **프로젝트 코드** | 코드 아이콘 | 기존 작업 기록 + 프리셋 |

**Select 드롭다운 옵션** (2개):

| 필드                       | 아이콘          | 데이터 소스          |
| -------------------------- | --------------- | -------------------- |
| **업무명 (Select 옵션)**   | 앱스토어 아이콘 | 기본값 + 사용자 추가 |
| **카테고리 (Select 옵션)** | 태그 아이콘     | 기본값 + 사용자 추가 |

### 옵션 숨기기

불필요한 옵션을 자동완성에서 숨길 수 있어요:

1. 숨길 옵션 체크
2. **"선택 항목 숨기기"** 버튼 클릭
3. 자동완성 목록에서 해당 옵션이 사라져요

### 숨긴 옵션 복원

1. **"숨긴 항목"** 섹션 펼치기
2. 복원할 옵션 선택
3. **"선택 항목 복원"** 버튼 클릭
